# Congrats

Luxury graduation congratulations platform — React + Vite + Supabase.

## Local development

```bash
npm install
cp .env.example .env   # add your Supabase keys
npm run dev
```

## Deploy (free)

1. Push to GitHub
2. Connect [Vercel](https://vercel.com) or [Netlify](https://netlify.com)
3. Set environment variables from `.env.example`
4. Add production URL in Supabase → Authentication → URL Configuration
